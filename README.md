# NightMoon
Based on ProtoHax

# How to Build
1. Run CloudBurstMC/Protocol publishToMavenLocal
2. You need to download our depository
3. Run ProtoHax publishToMavenLocal
4. Open ProtoHax-Android
5. Build Apk file
